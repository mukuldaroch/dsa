# Big O Summary

| Time Complexity           | Description                                           | Example                            |
| ------------------------- | ----------------------------------------------------- | ---------------------------------- |
| O(1) - Constant           | Time is constant regardless of input size.            | Array indexing.                    |
| O(log N) - Logarithmic    | Time grows logarithmically as input size increases.   | Binary search.                     |
| O(N) - Linear             | Time grows proportionally with input size.            | Searching in an unsorted list.     |
| O(N log N) - Linearithmic | Common in divide & conquer algorithms.                | Merge sort, quicksort.             |
| O(NÂ²) - Quadratic         | Time grows quadratically (nested loops).              | Bubble sort, selection sort.       |
| O(2^N) - Exponential      | Doubles with each increase in N.                      | Subset generation.                 |
| O(N!) - Factorial         | Grows factorially, extremely slow.                    | Traveling salesman problem.        |
| ------------------------- | ----------------------------------------------------- | ---------------------------------- |
